# Student Tracking Web Application
Created by Alex Cantrall, Solomon McKenley, Connor Cole, Nathanael Wolfe, and Chloe Fenner @ ETSU for CSCI-4250

# Project Overview
This project aims to build a student time tracking and communication web application using Blazor that allows students to accurately record their work hours and allow communication between instructors and students. The web app includes location bound clock-ins, a visual map, and a messaging system to ensure students are held accountable and supervised effectively.

# Vision Statement
To create a web application that allows students to accurately track their work hours, enables instructors to validate attendance through location based verification, and provide communication between instructors and students.

# Features
	- Allow students to clock in and clock out to track their time
	- Bind location on clock in, and display it with a pin on the map 
  	- Instructor -> Student and Student -> Instructor messaging
  Note: To check your messages, you have to go to the login screen first.
